    getPointAt

    • getPointAt(t: number): Point

      getPointAt returns a point on the line using parametric form: P(t) = P1 + t * direction Here, direction is the vector from P1 to P2, so t = 0 gives P1 and t = 1 gives P2.

      Parameters

      • t: number

        Parameter along the line. t=0 → p1, t=1 → p2, t=0.5 → midpoint between p1 and p2
